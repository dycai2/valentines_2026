# Valentines 2026

An interactive Valentine's Day website with animated hearts, playful buttons, and multi-page transitions.

## Pages

1. **index.html** - Asks "Will you be my Valentine?" with Yes/No buttons. The No button runs away when you hover over it.
2. **happy.html** - Celebrates with "Yippee!" then transitions to a dinner invite. The No button swaps to Yes on hover.
3. **happier.html** - Final page with confetti burst and a "You are now my valentine" message.

## Features

- Floating heart animations across all pages
- Smooth fade transitions between pages
- Playful, evasive "No" button mechanics
- Confetti burst effect on the final page
- Mobile responsive design
- Gwendolyn cursive font styling

## Tech Stack

- HTML / CSS / vanilla JavaScript
- No frameworks or dependencies

## Running Locally

Open `index.html` in a browser. No build step required.
